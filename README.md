# Спорт рядом — поиск спортивных объектов Москвы

Веб-приложение для поиска спортивных объектов в Москве с учётом доступной среды, видов спорта и местоположения.

## О проекте

Приложение помогает жителям Москвы находить подходящие спортивные объекты — стадионы, бассейны, спорткомплексы — с учётом индивидуальных предпочтений. Основная задача — сделать открытые данные города удобными для пользователя.

## Основные возможности

- **Поиск и фильтры** — по названию, виду спорта, району, доступности для маломобильных групп
- **Карточка объекта** — подробная информация: адрес, часы работы, контакты, карта с меткой
- **Избранное** — авторизованные пользователи могут сохранять понравившиеся объекты
- **Сравнение** — выберите 2–3 объекта и сравните их характеристики в таблице
- **Геопоиск** — найдите объекты рядом с вашим домом (геокодинг через OpenStreetMap)

## Технологии

- **Backend**: Python 3.11, Flask
- **База данных**: PostgreSQL
- **Frontend**: HTML, CSS
- **API**: Nominatim (OpenStreetMap)
- **Данные**: Портал открытых данных Москвы (data.mos.ru)
